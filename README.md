# zen-note

A simple and intuitive note-taking application. The app will feature a clean and user-friendly interface allowing users to quickly add, edit, and delete notes. The design will prioritize ease of use and minimalistic aesthetics, with a focus on functionality like search and organization capabilities.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/zen-note.git
cd zen-note
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
